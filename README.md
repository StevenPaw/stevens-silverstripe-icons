# stevens-silverstripe-icons

## Documentation

To use, simply set the icon in the class using the `stevens-icon-` prefix:

```php
    //In Elements:
    private static $icon = 'stevens-icon-news';

    //In Admins:
    private static $menu_icon_class = 'stevens-icon-news-admin';

    //In Pages:
    private static $cms_icon_class = 'stevens-icon-news-page';
```
